# react-bookstore
*Online bookstore* 

## Description

The web app of bookstore.

## Development

### Install dependencies
```
yarn
```

### Running the app locally

#### Server
```
yarn server
```

#### Client
```
yarn start
```
After running the project for server and client open your browser and go to http://localhost:3000
